# Recoding Rules and Baselines

This file documents how the raw data was processed for this analysis.

- **Recoding Rule (SadOrHopeless)**:
    - Success (Condition Met) = Code 1 (Yes)
    - Failure (Condition Not Met) = Code 2 (No)
- **Statistical Baseline**:
    - The null hypothesis baseline (p0) for this analysis is set at **0.50** (or 50%) to test if the proportion significantly differs from a random distribution.
- **Data Cleaning**: Missing values and non-responses were removed during the preprocessing stage to ensure data quality.
